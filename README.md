Number Guess Game 🎯
A simple Java console-based Number Guess Game where the player tries to guess a randomly generated number between 1 and 100.

Features
Random number generation
User input using Scanner
Hint system (Too High / Too Low)
Tracks number of attempts
Play again option
Win counter
Technologies Used
Java
Scanner Class
Random Class
How to Run
Clone the repository:

git clone https://github.com/your-username/your-repo-name.git
Navigate to the project folder:

cd your-repo-name
Compile the program:

javac NumberGuessGame.java
Run the program:

java NumberGuessGame
Game Rules
The computer generates a random number between 1 and 100.
Enter your guess.
The program will tell you if your guess is:
Too High
Too Low
Keep guessing until you find the correct number.
Choose whether to play another round.
